# 脚本仓库

这个仓库用于存放各种常用脚本，按语言分类管理。

## 目录结构

```
.
├── bash/           # Bash 脚本
├── python/         # Python 脚本
├── powershell/     # PowerShell 脚本
└── docs/           # 文档和说明
```

## 使用说明

### Bash 脚本
存放在 `bash/` 目录下，主要用于 Linux/macOS 系统自动化任务。

### Python 脚本
存放在 `python/` 目录下，跨平台脚本和数据处理工具。

### PowerShell 脚本
存放在 `powershell/` 目录下，主要用于 Windows 系统管理和自动化。

## 贡献指南

- 每个脚本应包含清晰的注释说明用途和使用方法
- 复杂脚本建议在 `docs/` 目录下添加详细文档
- 脚本命名使用小写字母和下划线，如 `backup_database.sh`
